# CodeAlpha Task 2 - Car Price Prediction

## Project Overview

This project is developed as part of the CodeAlpha Data Science Internship.

The objective of this project is to build a Machine Learning model that predicts the selling price of a used car based on different features such as manufacturing year, present price, kilometers driven, fuel type, selling type, transmission, and owner information.

## Dataset

Dataset Used: **car data.csv**

The dataset contains information about used cars, including:

Car Name
Year
Selling Price
Present Price
Driven Kilometers
Fuel Type
Selling Type
Transmission
Owner

## Machine Learning Model

Model Used:

**Linear Regression**

The model is trained using 80% of the dataset and tested on the remaining 20%.
